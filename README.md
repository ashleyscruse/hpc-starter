# HPC Starter

A gentle, hands-on intro to working on a high performance computing (HPC) system. No research, no heavy computing, just getting comfortable: log in, move around, edit a file, run it, and organize your data.

**Follow the walkthrough here:** https://ashleyscruse.github.io/hpc-starter/

## What you'll practice

1. Clone this repo onto the system
2. Look around the files and folders
3. Make your own folder
4. Edit a Python script with `nano`
5. Run your script (hello world)
6. Copy a dataset into your workspace

## Quick start

Once you have a terminal on Vista:

```bash
cd $WORK
git clone https://github.com/ashleyscruse/hpc-starter.git
cd hpc-starter
```

Then open the walkthrough and follow along.

## What's in here

| File / folder | What it is |
|---|---|
| `hello.py` | The script you'll edit and run |
| `data/flights_500.csv` | A small sample dataset to practice copying |
| `examples/welcome.txt` | A file to practice reading |
| `docs/` | The follow-along walkthrough (this is the website) |

Made for the MSCF summer cohorts. You can't break anything here. If you get stuck, ask.
